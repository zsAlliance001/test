# 番茄钟桌面应用 - 规格文档

## 1. 项目概述

- **项目名称**: Pomodoro Timer
- **类型**: 桌面应用 (Electron)
- **核心功能**: 番茄工作法计时器，支持工作/休息交替、统计记录、系统通知
- **目标用户**: 需要专注工作的用户

## 2. UI/UX 规格

### 窗口配置
- **窗口尺寸**: 320×480 像素 (固定，不可拉伸)
- **窗口样式**: 无边框，自定义标题栏
- **位置**: 屏幕居中

### 布局结构
```
┌─────────────────────────┐
│  [—][□][×]   番茄钟      │  ← 自定义标题栏 (32px)
├─────────────────────────┤
│                         │
│      25:00              │  ← 大号计时器显示 (主区域)
│                         │
│   ●●●●○○○○○○           │  ← 番茄数量指示器
│                         │
│   [开始]  [重置]        │  ← 控制按钮
│                         │
├─────────────────────────┤
│  🍅 工作 25分钟          │
│  ☕ 短休息 5分钟         │  ← 设置面板 (可折叠)
│  🌿 长休息 15分钟        │
│                         │
│  今日完成: 4 个          │  ← 统计信息
└─────────────────────────┘
```

### 视觉设计

**颜色调色板**
- 主色调: `#E74C3C` (番茄红)
- 次要色: `#2ECC71` (休息绿)
- 背景色: `#1A1A2E` (深夜蓝黑)
- 卡片背景: `#16213E` (深蓝)
- 文字主色: `#FFFFFF`
- 文字次色: `#A0A0A0`
- 强调色: `#F39C12` (金色)

**字体**
- 主字体: "Segoe UI", sans-serif
- 计时器数字: "Consolas", monospace, 72px, 加粗
- 标题: 16px
- 正文: 14px

**间距系统**
- 基础单位: 8px
- 内边距: 16px
- 元素间距: 12px

**视觉效果**
- 窗口圆角: 16px
- 卡片圆角: 12px
- 按钮圆角: 8px
- 标题栏渐变: 从 `#0F0F23` 到 `#1A1A2E`
- 卡片阴影: `0 4px 20px rgba(0,0,0,0.3)`
- 计时器脉冲动画: 当运行时，数字有轻微呼吸效果

### 组件状态

**开始/暂停按钮**
- 默认: 背景 `#E74C3C`，文字白色
- 悬停: 亮度+10%
- 运行中: 变为 "暂停" 按钮，背景变暗
- 禁用: 透明度 50%

**重置按钮**
- 默认: 背景透明，边框 `#A0A0A0`
- 悬停: 边框变白

**番茄数量指示器**
- 已完成: 实心红色圆点 `●`
- 未完成: 空心圆圈 `○`

## 3. 功能规格

### 核心功能

**计时功能**
- 工作计时: 默认 25 分钟
- 短休息: 默认 5 分钟
- 长休息: 默认 15 分钟 (每 4 个番茄后触发)
- 计时精度: 秒级
- 后台运行: 最小化到托盘继续计时

**控制功能**
- 开始: 开始/继续计时
- 暂停: 暂停计时
- 重置: 重置当前阶段到初始时间

**番茄计数**
- 每完成一个工作周期，计数 +1
- 每完成 4 个番茄后，自动切换到长休息
- 今日统计: 显示当天完成的番茄数

**系统集成**
- 系统通知: 计时结束时发送桌面通知
- 托盘图标: 最小化时显示托盘，点击恢复窗口
- 托盘菜单: 包含 开始/暂停、重置、退出 选项

### 用户交互流程

1. 启动应用 → 显示工作计时 25:00
2. 点击开始 → 计时开始，数字倒数
3. 计时结束 → 播放提示音 + 系统通知 → 自动切换到休息
4. 休息结束 → 播放提示音 + 系统通知 → 自动切换到工作
5. 每 4 个番茄后 → 强制长休息

### 数据持久化
- 使用 electron-store 保存设置和统计数据
- 保存内容: 各项时间设置、今日番茄数、统计日期

### 边界条件
- 应用关闭时正在运行的计时: 保存状态，下次启动询问是否继续
- 日期变更: 凌晨自动重置今日番茄数

- [ ] 托盘图标可见，右键有菜单
- [ ] 统计数据显示今日完成数
